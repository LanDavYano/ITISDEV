# Instructions to run
## Install Database Dependencies 
  - MongoDB Compass
  - MongoDB Community Server
  https://www.mongodb.com/products/self-managed/community-edition
  
## Seed Database
  - npm run seed
  - npm run db:check
  
## Install Needed Dependencies
  - npm install

## Run the local production build
  - npm run dev
  - open localhost
